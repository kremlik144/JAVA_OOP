    Напишите программу, позволяющую анализировать посещаемость студентов.
    Используйте паттерн MVP.

    Есть группа студентов. Для каждого студента есть журнал его посещаемости: 
    список для занятий и для каждой даты - посетил студент занятие или нет.
    - Создайте класс AttendanceService(сервис посещаемости), в котором храниться информация 
    обо всех студентах.
    - Создайте класс AttendanceView, который позволяет отображать
    студентов и их посещаемость.
    
    Создайте класс контроллера со следубщими функциями:
    - Распечатать всех студентов и посещаемость каждого в %
    - Распечатать студентов, отсовтировав их по убыванию посещаемости
    - Распечатать студентов с посещаемостью ниже 25% 




1. Запуск программ из файла Main.java

2. Модуль Presenter содержит публичный класс Controller,
    запускающий тот или иной сценарий действий, в зависимости от выбранного
    пользователем режима работы

3. Модуль Model берет всю работу на себя, содержит:
    а. публичный класс AttendanceService который:
        - генерирует группу студентов (метод generateGrup()) с заданным пользователем 
          номером группы 
        - метода getDataMode1() который собирает данные режима №1
        - методов getDataMode2() и sortedStudentsMode2() которые сортируют данные режима №2
        - методов getDataMode3() который собирает данные режима №3
    b. публичный класс StudentGroup который:
        - необходим для создания группы студентов 
    с. публичный класс Student который:
        - генерирует в конструкторе успеваемость студента, с заданными ФИ и дополнительно 
          в переменной attendancePercentage подсчитывает количество посещенных занятий 

4. Модуль Viev содержит класс AttendanceView который получает данные из 
   класса Controller и выводит их с помощью самодельных методов print() и mode4() 
   в консоль
